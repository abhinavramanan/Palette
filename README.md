# 🎨 Color Palette Generator

A beautiful, modern web application for generating and managing color palettes with various color harmony algorithms. Perfect for designers, developers, and anyone working with colors.

## ✨ Features

### 🌈 Color Harmony Generation
- **Analogous**: Colors that are next to each other on the color wheel
- **Complementary**: Colors opposite each other on the color wheel  
- **Triadic**: Three colors equally spaced around the color wheel
- **Monochromatic**: Different shades and tints of a single hue
- **Random**: Completely random color combinations

### 🛠️ Advanced Tools
- **💾 Save Palettes**: Store your favorite color combinations locally
- **📤 Export Options**: Export palettes in multiple formats:
  - CSS Variables
  - SCSS Variables  
  - JSON
  - Adobe ASE format
- **♿ Accessibility Checker**: Analyze color contrast ratios for WCAG compliance
- **📋 One-Click Copy**: Click any color to copy it to clipboard

### 🎯 Color Formats
Each color is displayed with multiple format options:
- **HEX**: `#FF5733`
- **RGB**: `rgb(255, 87, 51)`
- **HSL**: `hsl(14, 100%, 60%)`

## 🏃‍♂️ Quick Start

### Option 1: Direct Download
1. Download or clone this repository
2. Open `index.html` in your web browser
3. Start generating beautiful color palettes!

### Option 2: Local Server
```bash
# Clone the repository
git clone https://github.com/abhinavramanan/Palette.git
cd Palette

# Start a local server (Python 3)
python3 -m http.server 8000

# Or use Node.js
npx serve .

# Or use any other static server
```

Visit `http://localhost:8000` in your browser.

## 🎯 How to Use

### Generating Palettes
1. **Choose a harmony type** from the dropdown menu
2. **Click "🎨 Generate New Palette"** to create a new color combination
3. **Click any color** to copy it to your clipboard

### Managing Palettes
1. **Save palettes** you like by clicking "💾 Save Palette"
2. **Load saved palettes** by clicking the "Load" button in the saved palettes section
3. **Delete palettes** you no longer need

### Exporting Palettes
1. Click "📤 Export" to open the export modal
2. Choose your desired format (CSS, SCSS, JSON, or Adobe ASE)
3. Copy the generated code to your clipboard

### Accessibility Testing
1. Click "♿ Check Accessibility" to analyze color contrast ratios
2. View WCAG compliance ratings for all color combinations
3. Ensure your palette meets accessibility standards
